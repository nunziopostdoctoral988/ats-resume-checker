Important Limitations

This is a rule-based / heuristic engine, not a connection to any real employer's actual ATS software (Workday, Taleo, Greenhouse, etc). Results are directional guidance to help you improve your resume — not a guarantee of any real-world outcome.

We do not fabricate percentile rankings ("Top 5% of candidates") because we have no real aggregate dataset of other candidates to compare against — we use transparent A+–F score bands instead.

AI rewriting / cover-letter generation is not enabled by default because it requires connecting a paid LLM API key, which isn't configured in this build.

Privacy

Your resume and job description are processed entirely inside your browser using JavaScript. Files are never uploaded to any server.

Report history is stored only in your browser's local storage, on your own device. Clearing your browser data will erase it.
